# Screen List

각 화면은 6장의 기능 요구사항과 1:1로 매핑된다. Figma 프레임 이름은 이 표의 Screen ID를 그대로 사용한다.

---

## 인증 (Authentication)


| Screen ID   | 화면명  | 설명             |
| ----------- | ---- | -------------- |
| LOGIN-01    | 로그인  | 초기 사이트 접속 시 화면 |
| REGISTER-01 | 회원가입 | 회원가입 화면        |


## 홈 


| Screen ID | 화면명   | 설명                   |
| --------- | ----- | -------------------- |
| HOME-01   | 홈     | 로그인 후 메인 홈 화면        |
| HOME-02   | 알림 패널 | 알림 아이콘 클릭 시 뜨는 알림 패널 |


## 환율 탐색 (Rate Discovery)


| Screen ID | 화면명    | 설명         |
| --------- | ------ | ---------- |
| SRCH-01   | 환전소 지도 | 환전소 지도 페이지 |
| SRCH-02   | 환전소 상세 | 환전소 상세 페이지 |
| SRCH-03   | 통화 검색  | 통화 검색 페이지  |
| SRCH-04   | 환율 상세  | 환율 상세 페이지  |


## 예약 (Reservation)


| Screen ID      | 화면명      | 설명                               |
| -------------- | -------- | -------------------------------- |
| RSV-01         | 시간/금액 입력 | 시간/금액 입력 페이지                     |
| RSV-02         | 예약 확인    | 예약 확인 페이지                        |
| RSV-03         | 예약 상세    | 예약 상세 페이지                        |
| RSV-03-1       | QR 확인    | RSV-03에서 QR코드 버튼 클릭 시 QR 확인 화면   |
| RSV-03-2 POPUP | 취소 확인 팝업 | RSV-03에서 Cancel 클릭 시 뜨는 취소 확인 팝업 |
| RSV-03-2       | 취소 완료    | 팝업에서 취소 확정 시 최종 취소 완료 화면         |




## 내 예약 관리 (My Reservations)


| Screen ID | 화면명   | 설명                          |
| --------- | ----- | --------------------------- |
| MY-01     | 마이페이지 | 네비게이션 바의 Profile 클릭 시 마이페이지 |
| MYRSV-01  | 예약 내역 | 내 예약 내역 확인 페이지              |
| MYRSV-02  | 환전 기록 | 환전 기록 확인 페이지 (완료/취소)        |
| MYRSV-03  | 기록 상세 | 이전 기록 상세 페이지                |
