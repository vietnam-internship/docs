
---
ADM-1: Admin Login

The system will give exchange shops admin accounts(they don’t need to sign up)
Admin(exchange shops) can login with these accounts

---
ADM-2: Dashboard page

The admin can view the following things at the right side
today total users(reservation customers)
`#` of pending reservations today
currencies ranking(sorted by # of reservations today)
recent confirmed reservations list
The admin can access the left side that directed each pages.

---
ADM-03: Exchange Rates page

The admin can view the following things:
Base/Target currency pairs with Buy/Sell rates and fees that the admin set

The admin can perform the following actions:
Edit Buy/Sell rates and Fee percentages
Add a new currency pair and click "Save changes"
Navigate to other pages via the left sidebar

---
ADM-04: Inventory page

The admin can view the following things:
Selected exchange shop location
Currency list with stock balances and low stock status badges

The admin can perform the following actions:
Adjust currency stock balances
Save stock modifications by clicking "Adjust stock"
Navigate to other pages via the left sidebar

---
ADM-05: Reservations page

The admin can view the following things:
Reservation list with customer details and status badges (e.g., Pending, Cancelled)
Search bar and status filter tabs (ALL, Completed, Cancelled, Pending)

The admin can perform the following actions:
Search and filter reservations by status
View reservation details or navigate list pages via pagination
Navigate to other pages via the left sidebar

---
ADM-6-1: QR Scan - Checking page

The admin can view the following things:
Camera stream area for scanning QR codes
Simple guide instruction ("Please show the QR code")

The admin can perform the following actions:
Scan a customer's QR code using the camera to proceed to the next page
Navigate to other admin pages via the left sidebar menu



ADM-6-2: QR Scan - Confirmation page

The admin can view the following things:
Scanned customer's reservation details (name, ID, status, and exchange info)
Transaction details (amount, pickup location, date) and QR code status

The admin can perform the following actions:
Finalize the transaction by clicking the "Complete" button
Cancel or deny the transaction by clicking the "Reject" button
Navigate to other admin pages via the left sidebar menu



ADM-6-3: QR Scan - Reservation Complete page

The admin can view the following things:
"Reservation Complete" status message with a green check icon
Finalized transaction summary (Reservation number, Amount, Location, Date)

The admin can perform the following actions:
Return to the reservation list by clicking the "Back to list" button
Navigate to other admin pages via the left sidebar menu



ADM-6-4: QR Scan - Reservation Rejected page

The admin can view the following things:
"Reservation Rejected" status message with a red cross icon
Rejected transaction summary (Reservation number, Amount, Location, Date)

The admin can perform the following actions:
Return to the reservation list by clicking the "Back to list" button
Navigate to other admin pages via the left sidebar menu

The admin can perform the following actions:
Edit shop details including name, latitude/longitude, operating hours, and slot limits

ADM-07-02: Settings page

The admin can view the following things:
Form fields for Exchange Shop Information (Name, Location coordinates, Open/Close Time, and Available Reservation limit)
Current set values pre-filled in each input field (e.g., previous name, previous time)

The admin can perform the following actions:
Edit shop details including name, latitude/longitude, operating hours, and slot limits
Save the updated shop information by clicking the "Adjust stock" button
Navigate to other admin pages via the left sidebar menu
