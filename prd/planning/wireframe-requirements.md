# Wireframe 요구사항
**담당**: 기획자 B · **상태**: 초안 (멘토 템플릿 적용 전)

---

| 화면 | 필요 요소 |
|---|---|
| Home | 통화 선택, 금액 입력, 환율 비교 리스트 진입점 |
| Exchange Comparison | 환전소별 환율·거리·영업시간 리스트, AI 추천 정렬 토글 |
| Reservation | 방문 예정 시각 선택, 예약 요약, 만료 시간 안내 문구 |
| AI Recommendation | 추천 이유 요약(자연어), 추천 순위 표시 |
| My Reservation | 예약 상태(대기/완료/취소), 남은 만료 시간 카운트다운 |
| (직원용) 예약 확인 화면 | 예약자 정보, 신원확인 체크 버튼, 결제 완료 버튼 |
