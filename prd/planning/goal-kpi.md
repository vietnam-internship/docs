# 3. Goal & KPI
**담당**: 기획자 B · **상태**: 개정 완료

---

## 1. Goal (서비스 목표)

> "여행객이 신뢰할 수 있는 사설 환전소의 실시간 환율을 비교·예약하여 대기 없이 안전하게 환전하게 하며, 환전소에는 효율적인 재고 관리 및 오프라인 고객 유입 극대화의 가치를 제공함."

---

## 2. KPI (핵심 성과 지표)

| 구분 | KPI 지표 | 정의 | 목표치 |
| :--- | :--- | :--- | :--- |
| **플랫폼 활성화** | **환율 비교 이용률** | 전체 사용자 중 환율 비교 페이지 이용 비율 | **60% 이상** |
| | **예약 전환율 (Look-to-Book)** | 환율 상세 조회 후 예약 프로세스 시작 비율 | **20% 이상** |
| **퍼널 완료** | **예약 완료율** | 예약 프로세스 시작 대비 확정 예약 완료 비율 | **60% 이상** |
| | **누적 퍼널 완료율** | 앱 진입 후 최종 현장 수령까지의 전체 완료 비율 | **50% 이상** |
| **운영 건전성** | **노쇼율 (No-Show)** | 예약 완료 건 중 오프라인 미방문 취소 비율 | **30% 이하** |
| | **자발적 취소율** | 미방문 예약 건 중 2시간 만료 전 자발적 취소 비율 | **50% 이상** |
| **서비스 품질** | **환전소 예약 이행률** | 확정 예약 건 중 환전소 귀책 없이 정상 거래 완료된 비율 | **98% 이상** |
| | **대체 매칭 성공률** | 환전소 귀책 미이행 시 인접 환전소 대체 연결 및 수령 완료 비율 | **40% 이상** |
| **비즈니스 가치** | **평균 환전 규모 (ATV)** | 거래 완료된 예약의 건당 평균 환전 금액 | **지속 모니터링** |
