# 1. 경쟁 서비스(Reference) 조사

## 1.1 트래블월렛 (Travel Wallet)

### 서비스 개요
- 선불 충전형 외화 카드 서비스
- 46개 통화 지원
- 달러, 엔화, 유로 등 주요 통화 환전수수료 0%(100% 환율 우대)
- 목표 환율 예약 기능 제공

### 예약 프로세스
1. 약관 동의
2. 통화 선택
3. 목표 환율 입력
4. 환전 금액 입력
5. 예약 기간 선택
6. 예약 완료

### 주요 특징
- 환전 한도
  - 최소 : 10달러
  - 최대 : 200만원
  - 증액 신청 시 최대 300만원
- KYC(고객확인)
  - 신분증
  - 국내 휴대폰 인증
  - 국내 원화계좌 인증
- 2024년 기준
  - 누적 카드 발급 630만 장
  - MAU 약 76.8만 명

### 우리 서비스 적용 포인트
- 목표 환율 예약 UX는 참고할 가치가 있음
- 하지만 트래블월렛은 앱 내에서 환전 및 결제까지 완료되는 구조
- 우리 서비스는 **오프라인 환전소 방문 수령**이 핵심이므로
  - 예약 만료 시간
  - 노쇼(No-show) 정책
  - 재고 관리 정책
  등을 별도로 설계해야 함

## 1.2 은행권 인터넷 환전 (하나·신한·국민·우리은행)

### 특징
- 인터넷 또는 모바일 환전 시 최대 90% 환율 우대
- 은행마다 수령 가능 지점과 당일 수령 가능 여부가 다름
- 은행연합회에서 은행별 환율 비교 서비스 운영

### 우리 서비스 적용 포인트
- 기존 비교 서비스는 **은행만 비교 가능**
- 우리 서비스는 **사설 환전소까지 포함한 환율 비교**를 제공하여 차별화 가능
