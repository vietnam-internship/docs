# 2. 법률 및 규제 조사

## 2.1 환전업 등록 의무

### 관련 법령
- 외국환거래법 제8조 제3항
- 환전업은 관할 세관장(관세청)에 등록해야 운영 가능

### MVP 적용 방향
- Toy Project 단계에서는 등록 불필요
- PRD에는 **실서비스 전환 시 환전업 등록이 필요함**을 명시

---

## 2.2 개인 환전 한도 및 신고 기준

### 실명 확인
- 100만원 초과 환전 시 신분증 확인 필요

### 자동 통보
- 동일 날짜 기준 미화 1만 달러 초과 환전 시 국세청 자동 통보

### 출국 시 신고
- 미화 1만 달러 초과 휴대 시 세관 신고 의무

위반 시
- 3만 달러 이하 : 위반 금액의 5% 과태료
- 3만 달러 초과 : 1년 이하 징역 또는 1억원 이하 벌금

### 서비스 적용 방향
MVP에서는
- 1만 달러 이상 예약 기능 제외
- 또는 사전 안내 문구 제공

---

## 2.3 고객확인의무(KYC / CDD)

### 대상
환전업자는 특정금융정보법상 고객확인의무 적용 대상

### 대상 거래
계좌 없이 이루어지는 일회성 금융거래 중
약 미화 1만 달러 이상 거래

### 확인 항목
- 실명
- 주소
- 연락처

### 강화 고객확인(EDD)
고위험 고객은 추가 확인 필요

### 위반 시
- CDD : 3천만원 이하 과태료
- EDD : 1억원 이하 과태료

### 서비스 적용 방향

2단계 인증 구조

**예약 단계**
- 휴대폰 본인인증

**방문 수령 단계**
- 환전소 직원이 신분증 확인

추가 정책
- 일정 금액 이상 예약 시
  - "방문 시 신분증을 반드시 지참해주세요." 안내 제공
