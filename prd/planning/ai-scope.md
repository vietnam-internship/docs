# AI 기능 범위
**담당**: 기획자 B · **상태**: 초안

---

**기본 원칙**: AI는 메인 기능이 아니라, 환율 비교 결과 위에 얹는 **보조 추천 기능**으로 한정

| 항목 | 정의 |
|---|---|
| AI가 하는 일 | 사용자의 조건(방문 가능 시간, 환전 금액, 위치)을 반영해 환전소 후보를 **점수화해 정렬**해주는 추천 |
| AI가 하지 않는 일 | 미래 환율을 예측하거나 "지금 환전하세요"를 확정적으로 조언하는 것 (법적/신뢰 리스크 있음, 업계 정확도도 70% 수준에 불과 — [competitor-research.md](./competitor-research.md) 코어16 사례 참고) |
| 사용 모델(예시) | 별도의 예측 모델이 아니라, 환율·거리·영업시간·재고 등 정형 데이터를 조합한 규칙 기반 스코어링 + LLM은 결과를 자연어로 요약하는 역할 정도로 한정 |
| KPI(초안) | AI 추천 결과 클릭률 60% 이상 / AI 추천 상위 3개 중 실제 예약 전환율 30% 이상 |

> "예측 정확도 90%"보다는 "추천이 실제 선택으로 이어지는 비율"을 KPI로 삼는 게 더 현실적임 (업계 최고 수준 AI 환율예측 정확도가 66~70%대)
