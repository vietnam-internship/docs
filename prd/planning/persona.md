## Tourist / 유학생 (핵심 사용자)
- **목적**: 출국/입국 전 유리한 환율로, 대기 없이 원하는 시간에 환전하고 싶음
- **Pain Point**
  - 환율이 하루에도 여러 번 바뀌어 "지금이 환전 타이밍인지" 판단하기 어려움
  - 공항 환전은 편리하지만 수수료가 비쌈
  - 사설환전소는 환율이 좋지만 위조지폐·부족지급 등 신뢰 문제가 있고, 발품을 팔아야 함
  - 은행 인터넷환전은 우대율이 높지만 수령 가능 지점·시간이 제한적
- **행동**: 여행 1~2주 전부터 환율을 검색하며 타이밍을 살핌, 여러 앱/은행 환율을 비교, 명동 등 사설환전소 밀집 지역을 직접 검색

## Exchange Shop (환전소 직원)
- **목적**: 예약 고객을 효율적으로 응대하고, 재고를 낭비 없이 운영하고 싶음
- **Pain Point**
  - 예약 시스템이 없으면 방문객이 몰릴 때 대기가 길어짐
  - 노쇼가 잦으면 특정 통화 재고가 묶여 다른 손님을 못 받음
  - 신원확인(KYC) 절차를 누락하면 법적 리스크가 있음
- **행동**: 예약 확인 → 방문객 신원확인(신분증) → 결제 → 환전 완료 처리, 재고 현황을 수시로 확인

## Admin (플랫폼 운영자)
- **목적**: 여러 환전소의 데이터(환율, 재고, 예약 현황)를 관리하고 이상 거래를 모니터링하고 싶음
- **Pain Point**
  - 환전소별 우대율 데이터가 수기로 들어오면 최신성 유지가 어려움
  - 고액 예약(법적 신고 기준 근접) 발생 시 별도 확인이 필요함
- **행동**: 환전소 등록/관리, 우대율 데이터 검수, 예약 통계 모니터링
