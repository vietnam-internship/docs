# AI 모델 기술 검토

## 1. 검토 목적
AI 환전 시점 추천 기능은 서비스 핵심 기능(실시간 환율 비교 + 예약)을 **보조**하는 참고 정보 제공용이며, 투자 자문이 아니다.

## 2. 모델 비교

| 모델 | 장점 | 단점 | MVP 적용 |
| --- | --- | --- | --- |
| 이동평균 규칙 모델 | 구현 쉬움, 안정적, 설명 가능 | AI 모델로 보기 어려움 | **폴백 로직** |
| Linear Regression | 구현 간단, 계수로 설명 가능, 적은 데이터에도 적용 가능 | 비선형/장기 패턴 반영 어려움 | **기본 모델** |
| Ridge Regression | 과적합 완화 | 해석이 약간 복잡 | 대안 (데이터 적거나 변동성 큰 경우) |
| ARIMA | 전통 시계열 예측에 적합 | 튜닝/정상성 검토 부담 | 향후 고도화 |
| LSTM | 장기 패턴 학습 가능 | 데이터·학습시간 많이 필요, 설명력 낮음 | 제외 |

**평가 항목별 비교**

| 항목 | Baseline | Linear Regression | LSTM/GRU | ARIMA |
| --- | --- | --- | --- | --- |
| 설명 가능성 | 높음 | 높음 (계수 기반) | 낮음 (블랙박스) | 중간 |
| 과적합 리스크 | 없음 | 낮음 (Ridge로 보완 가능) | 높음 | 중간 |
| 재학습 비용 | 없음 | 낮음 | 높음 (GPU 필요 가능) | 중간 |

→ MVP 단계에서 중요한 건 **설명 가능성 · 낮은 과적합 · 낮은 재학습 비용**이므로 Linear Regression이 적합.

## 3. MVP 기본 모델: Linear Regression
- 구현 난이도 낮음 → 개발 일정에 적합
- 회귀 계수로 추천 근거 설명 가능
- 적은 데이터로도 적용 가능 (초기 서비스 단계에 유리)
- AI는 보조 기능 → 복잡한 모델보다 안정성·설명력 우선

## 4. 폴백 로직 (이동평균 기반)
- 현재 환율 < MA - 1σ → **환전 고려**
- 현재 환율 > MA + 1σ → **대기 고려**
- 그 외 → **보통**

## 5. 향후 고도화 (데이터 축적 후)
Ridge Regression → Random Forest → ARIMA → LSTM/GRU 순으로 검토 확대
